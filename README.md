# Peace-Makers

Connect and discover lesser-known NGOs worldwide.

**Peace-Makers** is a website that highlights underrepresented NGOs and keeps their listings up to date, making it easier for people to discover and support them.

---

## Features
- **NGO Listings** – Curated collection of NGOs from around the world.  
- **Interactive UI** – Smooth navigation and responsive design.  
- **Multimedia Support** – Audio and video elements for better engagement.  
- **Always Updating** – Continuously adds new NGOs to keep the site relevant.  

---

## Tech Stack
**Frontend:** HTML5, CSS3, JavaScript  
**Backend:** None (Static site, deployed on Netlify)
